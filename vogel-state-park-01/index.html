Vogel State Park

Photography 9

So now that we have officially moved to Georgia (I should post about this at some point), we decided to take a break from unboxing/unpacking and visit one of the local parks; Vogel State Park.

Seems like a very nice little park. It was very crowded so it has quite the following, but I can easily see it as someplace where we could walk the dogs or just go relax for a couple hours.

It has fishing, hiking, boating and cabins..etc.. The whole nine yards! It Even has a wilderness/primitive campground area which I thought was pretty cool.. Very beautiful area.
